# **App Name**: IPL Insights Hub

## Core Features:

- User Authentication: Secure user registration and login with email and password using JWT.
- Team Statistics Display: Showcase team-specific statistics, match wins and losses, and key player contributions, presented in a clean, card-based UI.
- Match Results Table: Provide a sortable and filterable table of all IPL matches, including season, teams involved, and the match results. Supports pagination for efficient data loading.
- Player Statistics: Aggregate individual player stats like runs scored, wickets taken, and batting averages. Implement filters to narrow down the data by season or team.
- Data Insights Tool: Use LLM to identify interesting correlations in IPL data (e.g., weather conditions affecting scores), incorporating the use of statistical tools in making this determination.
- Admin Data Refresh: Enable admin users to trigger a refresh of the IPL dataset, which is then re-processed to update statistics.

## Style Guidelines:

- Primary color: Saturated violet (#9400D3), reminiscent of the IPL's royal associations and excitement.
- Background color: Light, desaturated lavender (#F0E6EF) for a premium feel.
- Accent color: Bright cyan (#00FFFF), for interactive elements and highlights to draw attention.
- Headline font: 'Playfair', a modern serif for elegant headings. Body Font: 'PT Sans' (sans-serif) for body text and table displays.
- Use a consistent set of minimalist icons for navigation and data representation. Use icons to represent statistics like runs, wickets, and wins.
- Employ a card-based layout for displaying team and player statistics, and a clean tabular layout for match data.
- Subtle animations, like transitions and loading spinners, to enhance user experience without being intrusive.
